Como trabalho na Caixa na área de segurança, preciso desofuscar logs de ataques de phishing para conseguir entender os ataques e poder previní-los.

Sou dono de um hospital e, com a LGP, todo o banco de dados foi ofuscado. Preciso que alguns dados sejam desofuscados para que eu possa apresentá-los ao paciente relacionado.

Sou dono de uma empresa antivírus e os hackers começaram a empregar uma variedade de ferramentas para ofuscar o código subjacente de seu software para evitar a detecção com base em assinatura e dificultar as análises estáticas. Preciso que esse código seja desofuscado, permitindo, assim, que o antivírus detecte os comandos e verifique se fazem parte de malwares, entre outros.
